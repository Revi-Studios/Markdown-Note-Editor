# Project: Notemark editor

## How does it work?:

**Features:**

- Formats _current line_ on every input
- Formats to _markdown mix_

- When hovering over a formatted text, the _formatting symbols_ are shown

- Pressing on formatted checkmarks _checks_ them in text

- Copy as _normal text_
